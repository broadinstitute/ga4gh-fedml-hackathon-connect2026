# Overview

Here we contemplate a "Unit Cost Profile" for articulating the costs associated with each node within the federation. In the prelude to the hackathon, discussions on this topic surfaced the following items/challenges for consideration:

- egress costs
- data costs
- compute costs
- personnel costs
- variance across nodes: node size, node contribution to model accuracy, and node time to contribution
- complexity cost of the job (number of parameters x data size)

Note: Central server costs for aggregation etc. can be considered a unique case of a unit and should be considered if the federated system includes one.
